# Oasis — Modern Furniture E-commerce

A modern furniture store interface built with Vue 3 and Nuxt 4.

## Design

Based on:

[Oasis — Modern & Contemporary eCommerce UI Kit](https://www.figma.com/community/file/1384158689157076474)

The design was adapted with custom features and implementation improvements.

## Requirements

- Node.js >= 22
- pnpm >= 11

## Installation

Clone the repository:

```bash
git clone https://github.com/daisy40/oasis
```

Install dependencies:

```bash
pnpm install
```

Start development server:

```bash
pnpm dev
```
